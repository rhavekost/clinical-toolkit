# MME Calculation Guide (CDC Conversion Factors)

## Step 1: Calculate Daily Dose

For each opioid:
- Daily dose (mg/day) = dose per unit * units per day

## Step 2: Apply Conversion Factor

MME per opioid = daily dose (mg/day) * conversion factor

## Step 3: Sum Total MME

Total daily MME = sum of all opioid MME values

## CDC Conversion Factors (Common)

| Opioid | Conversion Factor |
|--------|-------------------|
| Codeine | 0.15 |
| Hydrocodone | 1.0 |
| Hydromorphone | 5.0 |
| Morphine | 1.0 |
| Oxycodone | 1.5 |
| Oxymorphone | 3.0 |
| Tapentadol | 0.4 |
| Tramadol | 0.1 |

## Special Cases

- **Fentanyl transdermal:** use mcg/hr * 2.4 = MME/day
- **Methadone:** conversion is complex and varies by dose; do not use a single factor for conversion between opioids

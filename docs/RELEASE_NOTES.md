# Release Notes

## 2.0.0

### Added
- New open‑licensed screening tools: K6/K10 (distress) and WHO‑5 (well‑being).
- Consensus Sleep Diary skill with compliant access guidance and summary workflow.
- Opioid MME calculator with CDC conversion factors and cautions.
- New non‑validated interview/checklist tools: bipolar risk interview, functional impairment interview, eating risk checklist, cognitive concerns intake.
- Expanded outcome tracking template with sleep diary adherence and new measures.

### Changed
- Motivation assessment now provides URICA access guidance (not instrument items).
- Updated clinical references with distress cutoffs, sleep pathway, and medication safety notes.
- Updated manifests and README to reflect new tools and licensing constraints.

### Removed
- Restricted or unclear licensing instruments were excluded to maintain open‑source compliance.
